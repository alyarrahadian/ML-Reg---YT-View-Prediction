# ML-Reg---YT-View-Prediction
This task focuses on predicting YouTube video views using statistical features and metadata associated with each video. The dataset contains various attributes such as engagement metrics, publishing details, and textual features. Mainly will be using regression and analysis interpretation available on indonesian.

## Data
Data is from : https://www.kaggle.com/datasets/datasnaek/youtube-new/data?select=CAvideos.csv

## Columns
| Column Name            | Description                                   |
| ---------------------- | --------------------------------------------- |
| trending_date          | Date when the video became trending           |
| title                  | Title of the video                            |
| channel_title          | Name of the YouTube channel                   |
| category_id            | Encoded category of the video                 |
| publish_time           | Time when the video was published             |
| tags                   | Tags used in the video                        |
| views                  | Number of views (target variable)             |
| likes                  | Number of likes                               |
| dislikes               | Number of dislikes                            |
| comment_count          | Number of comments                            |
| comments_disabled      | Whether comments are disabled                 |
| ratings_disabled       | Whether ratings (likes/dislikes) are disabled |
| video_error_or_removed | Whether the video is unavailable or removed   |
| description            | Description of the video                      |
| No_tags                | Number of tags used                           |
| desc_len               | Length of the video description (in words)    |
| len_title              | Length of the video title (in words)          |
| publish_date           | Date when the video was published             |
